# Nhà hàng Golden Taste

Hệ thống quản lý nhà hàng (F&B) — Node.js, MySQL, web SPA.

## Chạy nhanh

```powershell
cd server
npm install
# File .env: DB_NAME=ql_golden_taste
npm run setup-db
npm start
```

Mở http://localhost:3001 — đăng nhập `nv002@goldentaste.vn` / `password123`

## CSDL

- Schema: `database/golden_taste/` (theo `docs/QL_cuahanggoldentaste.sql`)
- Truy vấn mẫu: `docs/TRUY-VAN-MAU.sql`
